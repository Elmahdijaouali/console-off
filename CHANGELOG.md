# Changelog

All notable changes to the `console-off` package will be documented in this file.

## [1.0.0] - 2026-02-06

### Added
- Initial release
- Auto-detection of production environment
- Support for all major frameworks (React, Next.js, Vue, Angular, Express)
- Manual disable/restore console functions
- Option to exclude specific console methods
- Custom environment check function
- Force disable option
- Full TypeScript support
- Comprehensive documentation and examples

### Features
- Disables 20+ console methods
- Zero dependencies
- Lightweight (~2KB)
- Works in Node.js and browser environments
- Compatible with CommonJS and ES modules
